HTML

to give the content structure.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

to give the content style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding insicates how much space we want around the content inside an element.
  • You can create a class to more easily create similar looking sections whilit shorting code.
  • You can leave notes on style.css code (example: /*note goes here*/) which can better explain code and leave important information

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

to handle logic and allow users to interact with a website.
  • A variable is a named container that allows us to store out data in code.
  • Control flow is the order in which a computer executes code in a script.
